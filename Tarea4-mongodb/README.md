# Tarea 4 – Almacenamiento y Consultas de Datos en Big Data

## Descripción
Este repositorio contiene el desarrollo de la Tarea 4, donde se implementa un caso de estudio usando MongoDB.  
Incluye:
- Diseño del modelo de datos orientado a documentos  
- Inserción de 100 documentos generados dinámicamente  
- Consultas CRUD  
- Consultas de agregación  
- Análisis de resultados  

## Tecnologías utilizadas
- **MongoDB**
- **mongosh**
- **JavaScript (para scripts de mongo)**

## Archivos incluidos

### `script_insert.js`
Script encargado de:
- Crear la base de datos `ecommerce_db`
- Crear la colección `products`
- Generar automáticamente 100 documentos
- Insertarlos en la colección

### `queries_mongodb.js`
Contiene:
- Consultas CRUD
- Consultas con filtros avanzados
- Operadores de actualización
- Pipelines de agregación (group, sort, avg, sum)
